# AOS Widget Desktop

### AOS-WD v39.0.0 - almost the best lubricant for your AOS dev

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
